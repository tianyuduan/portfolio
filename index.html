Skills

FishPx

JavaScript React.js Redux Ruby/Rails PostgresSQL HTML5 CSS3 Material-UI

FishPx is a full-stack single page web application inspired by 500 px. FishPx allows users to navigate through tons of oceanic photography, upload their own photos to the website, and browse images via specific categories. It features a secure user authentication using Bcrypt.

ConneQt

JavaScript React-Native Express Node.js HTML5 MongoDB

ConneQT is a mobile application that is designed to improve the process of exchanging contact information. We created ConneQt to aid individuals in networking. The process is simple. To get ConneQted, simply scan another user's QR code and their contact information will appear in your personal account.

Flappy Fish

HTML5 Canvas JavaScript

FlappyFish is an infinite runner type game based on FlappyBird. It is a side-game scroller game where the user controls a fish. The purpose of the game is to fly between rows of rocks without hitting them. FlappyFish was built with HTML 5 canvas. The entire game was constructed using vanilla JavaScript.

About Me

I love learning. Whether its algorithims, programming languages, new frameworks, new technologies, working on improving yourself, the accumulation of all these pieces help me become the software developer I am today.

It all started with just a series of computer science courses. I quickly discovered the joy of absorbing an inexhaustable fountain of knowledge. Soon later, I decided to accelerate my learning process by attending App Academy, an 1000 hour extensive bootcamp. I learned to build dynamic web and mobile apps using JavaScript, React/React-Native, Redux, Ruby on Rails, Node.Js, Express, HTML5, CSS5, MongoDB and SQL

I think one of the greatest things about programming is bringing the manifestation of ideas into existence. Programmers break the barrier between conceptualization and a tangible, functional product. We can take what we learned, put pieces together, overcome challenges we face and craft a product to directly impact our world today.

I think that in itself is extremely gratifying.
